# Independent CAD UI implementation policy

This is an engineering policy, not legal advice. Obtain qualified German/EU counsel before a high-risk commercial launch.

## Permitted research and implementation approach

- Observe publicly available documentation, tutorials, and videos to understand functional behavior, user expectations, interaction sequences, and interoperability requirements.
- Implement those ideas independently with CAD File Lab source code, architecture, terminology where practical, branding, colors, icons, and visual expression.
- Keep dated internal documentation of sources consulted and the independent implementation decisions.
- Use official documentation as behavioral acceptance criteria, not as content to reproduce.

EU Directive 2009/24/EC states that copyright protects the expression of a computer program, while ideas and principles underlying program elements, including interfaces, are not protected under that Directive. It also recognizes observation, study, or testing of functioning by a lawful user under its conditions. The CJEU's SAS Institute decision likewise distinguishes program functionality from protected expression.

## Prohibited copying and branding

- Do not copy SOLIDWORKS source code, binaries, icons, logos, screenshots, manuals, video frames, sound assets, or proprietary visual artwork.
- Do not trace or reproduce SOLIDWORKS icons, dialogs, or screen layouts pixel-for-pixel.
- Do not use SOLIDWORKS or a confusingly similar term in the product name, domain, logo, or primary branding.
- Do not imply affiliation, certification, endorsement, compatibility approval, or partnership without written authorization.
- Do not imitate the SOLIDWORKS logo design scheme or use its registered logo as a product design feature.
- Do not bypass access controls, license controls, technical restrictions, or confidentiality duties to study the product.

SOLIDWORKS identifies its name and logos as Dassault Systèmes trademarks and publishes restrictions on unauthorized logo use and confusing affiliation.

## Product differentiation requirements

- Retain the CAD File Lab name and distinct visual identity.
- Use original icons and original copy.
- Adapt functional CAD conventions to the browser and local-processing privacy proposition.
- Where this repository mentions SOLIDWORKS, use it only nominatively in internal comparison documentation or factual compatibility/help text reviewed before publication.
- Run a trademark, design-right, copyright, patent, and unfair-competition review with counsel before monetized launch.
