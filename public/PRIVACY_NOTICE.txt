Application processing
----------------------

CAD File Lab processes imported CAD files inside the visitor's browser. The
application does not upload model contents, project files, feedback text, or
recovery data to an application server. Recovery data is stored in the
browser's IndexedDB database on the visitor's device and can be erased from
the application.

The application has no analytics, advertising, account system, or application
cookies. The feedback tool only copies text to the visitor's clipboard; it
does not transmit the text.

Hosting data
------------

The hosting and network provider may necessarily process request metadata,
including IP address, requested URL, timestamp, user agent, and security
signals, to deliver and protect the site. That infrastructure processing is
separate from CAD model processing. Consult the Cloudflare privacy policy for
its current practices: https://www.cloudflare.com/privacypolicy/

The purpose is reliable and secure delivery of the website. The operator's
legal basis for necessary request-log processing is the legitimate interest in
providing and protecting the service (Article 6(1)(f) GDPR). Cloudflare may
process data outside Germany under the safeguards described in its current
privacy and data-processing terms. Retention of infrastructure logs follows
the configured Cloudflare service and security settings; contact the operator
for information about a specific request.

Device storage
--------------

IndexedDB recovery storage is used only when the visitor enables or uses the
local recovery feature. It is necessary to provide the requested browser-local
save/recovery functionality and is never read by the application server. The
visitor can erase it using "Clear local recovery data" in the project panel or
through browser site-data controls.

Third-party links
-----------------

The notices and source-offer documents contain links to third-party source and
license sites. Opening those links sends a normal web request to that provider.

Rights
------

Subject to applicable law, data subjects may request access, rectification,
erasure, restriction, portability, or objection concerning personal data
processed by the operator. Requests can be sent to admin@cadfilelab.com. Data
subjects may also complain to a competent supervisory authority, including the
State Commissioner for Data Protection and Freedom of Information of Baden-
Wuerttemberg: https://www.baden-wuerttemberg.datenschutz.de/

No automated decision-making or profiling is performed by the application.
