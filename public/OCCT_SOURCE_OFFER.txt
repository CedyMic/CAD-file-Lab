Rebuilding and replacement
--------------------------

Use the binding YAML and additional C++ code under
packages/replicad-opencascadejs/build-source at the RepliCAD release commit.
Run them with the immutable build image above. The image pins OCCT and
Emscripten; using its digest prevents the former floating latest tag from
changing the build environment.

CAD File Lab loads /occt-kernel.json at runtime. Host a compatible rebuilt
WASM file on the same origin and set wasmUrl to its relative path. Detailed
instructions are in docs/OCCT_REPLACEMENT.md in the application source.

No warranty is provided for OCCT, OpenCascade.js, or replacement kernels.
